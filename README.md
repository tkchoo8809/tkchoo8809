## 📁 Featured Projects

### 🔍 Document Intelligence RAG Prototype
An open-source pipeline for extracting structured data from scanned documents using fine-tuned vision language models and Retrieval-Augmented Generation (RAG).

**Tech Stack**: Python, PyTorch, Qwen2.5-VL, ColQwen2, Qdrant, Pydantic AI

**Key Features**:
- Multi-modal document processing (images & PDFs)
- Efficient fine-tuning with 4-bit quantization
- Vector-based retrieval with metadata filtering
- Conversational agent interface
- Local LLM integration (Ollama)
